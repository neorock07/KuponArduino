# Program Kupon RFID KX

Program ini digunakan untuk keperluan pembuatan alat transaksi kupon.

-----

## Component 

ESP32-WROOM
ETHERNET MODULE W5500
AMPLIFIER MAX98357A
RFID SENSOR MFRC522
SPEAKER 4 ohm
LCD SPI TFT 4 INCH
